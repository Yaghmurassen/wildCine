# wildcine

> A Vue.js project

## Build Setup
Clône le depot

    git clone git@github.com:Yaghmurassen/wildCine.git

# install dependencies
on npm install à la racine :

    cd wildCine && npm i

lancer les commandes suivantes :

    cd api/ && node server

Puis pour finir :

    cd ../ && npm run dev


Voilà bonne visite ! =)
